# graphed-corpus

Ground-truth **requirements + runnable functional fixtures** for `graphed`, distilled from real HEP
analyses (milestone M0.5). Holds the **Required Operations Catalog**
(`docs/requirements/ops_catalog.md`), the **Canonical Analyses** with stored reference histograms
(`tests/corpus/`), and the **graph-bloat note** that sets M4's reduction targets. **No framework
code** — requirements and reference data only.

Part of the [`graphed-org`](https://github.com/graphed-org) project; see
[`graphed-project`](https://github.com/graphed-org/graphed-project) for root guidance and the plan.

Status: **M0.5 — bootstrapping.** See `CLAUDE.md` for the distilled specification.
